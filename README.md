# Simulador de Lanzamiento de Dados

Aplicación Python que simula el lanzamiento de dados, desarrollada como práctica de CI/CD en el módulo 17 del curso de Ingeniería de Datos de EOI.

---

## Descripción

Esta aplicación simula **5 lanzamientos de un dado de 6 caras** y muestra el resultado de cada tirada por consola, con una pausa de 5 segundos entre cada lanzamiento.

Hace uso de la librería [`dice`](https://pypi.org/project/dice/) para generar los resultados de forma aleatoria siguiendo la notación estándar de dados de rol (`NdM`, donde `N` es la cantidad de dados y `M` el número de caras).
